        <!-- Descrição principal -->
        <div style="background: #eef2fa; border-left: 5px solid var(--azul); padding: 20px; border-radius: 12px; margin-bottom: 32px;">
            <p style="margin: 0; font-size: 1rem; line-height: 1.5;">
                <strong>Gerador de Documentos Administrativos</strong> é uma ferramenta web gratuita criada para automatizar a criação de peças essenciais ao ciclo de contratações públicas: 
                <strong>DFD (Documento de Formalização da Demanda)</strong>, 
                <strong>ETP (Estudo Técnico Preliminar)</strong> e 
                <strong>Termo de Referência</strong>.
                Com ela, servidores e gestores ganham agilidade, reduzem erros manuais e garantem a padronização exigida pela Lei 14.133/2021.
            </p>
        </div>

        <!-- Grid com como funciona e benefícios -->
        <div style="display: grid; grid-template-columns: 1fr 1fr; gap: 24px; margin-bottom: 32px;">
            <div>
                <h3 style="color: var(--azul); margin-bottom: 12px;">⚙️ Como funciona</h3>
                <ol style="margin: 0; padding-left: 20px; color: var(--texto); line-height: 1.5;">
                    <li>Escolha o tipo de documento (DFD, ETP ou TR)</li>
                    <li>Preencha os campos do formulário (objeto, justificativa, valores etc.)</li>
                    <li>Clique em “Gerar” – o sistema insere os dados no modelo</li>
                    <li>O arquivo <code>.docx</code> é baixado pronto para uso</li>
                </ol>
            </div>
            <div>
                <h3 style="color: var(--azul); margin-bottom: 12px;">✅ Benefícios para o serviço público</h3>
                <ul style="margin: 0; padding-left: 20px; color: var(--texto); line-height: 1.5;">
                    <li><strong>Agilidade:</strong> documentos gerados em segundos</li>
                    <li><strong>Padronização:</strong> todos os documentos seguem o mesmo layout</li>
                    <li><strong>Conformidade:</strong> modelos alinhados à legislação</li>
                    <li><strong>Segurança:</strong> 100% no navegador – nenhum dado é armazenado</li>
                    <li><strong>Flexibilidade:</strong> use os templates padrão ou faça upload dos seus próprios modelos <code>.docx</code></li>
                </ul>
